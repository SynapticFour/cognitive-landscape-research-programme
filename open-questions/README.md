# Open questions

Explicit register of **unresolved** programme questions. Keeping uncertainty visible is a CLRP epistemic commitment (CLRP-004).

## Format

Each question is a Markdown file or section with:

- **ID:** OQ-NNN
- **Question**
- **Why it matters**
- **Blocking** (what work is stalled)
- **Related documents**
- **Status:** Open | In progress | Resolved (link)
