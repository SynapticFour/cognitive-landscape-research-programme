# Validation (CLRP-VR)

Empirical evidence reports: psychometric studies, adaptation evaluations, and programme evidence summaries.

## Series ID

`CLRP-VR-YYYY-NNN`

## Standards

All validation reports follow [CLRP-006](../clrp/CLRP-006-evidence-and-validation.md) and [templates/validation-report-template.md](../templates/validation-report-template.md).

## Honest status

Validation reports must state:

- Current validation tier (T0–T4)
- What claims are **not** supported
- Locale scope

## Relationship to implementation docs

Implementation repositories (e.g. PCMS `VALIDATION_PROTOCOL.md`) should **shrink** to pointer + local status, with evidence living here as CLRP-VR.
