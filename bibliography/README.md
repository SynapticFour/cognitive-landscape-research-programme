# Bibliography

Shared BibTeX for CLRP documents, technical reports, and validation reports.

## Usage

1. Add entries to [references.bib](references.bib)
2. Cite by key in Markdown documents
3. Do not duplicate full bibliographies in every CLRP spec — link here

## Conventions

| Field | Convention |
|-------|------------|
| Key prefix | `clrp_` for programme docs; author-year otherwise |
| DOI | Include when available |
| URL | Prefer persistent links |

## Citation style

Repository default for human-readable docs: **APA 7** (see [docs/citation/citation-guide.md](../docs/citation/citation-guide.md)).

## Contributing

PR with new `@article`, `@book`, `@misc` entries and context for where used.
