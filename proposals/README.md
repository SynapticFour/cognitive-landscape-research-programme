# Proposals (CLRP-P)

Programme extensions awaiting acceptance into `clrp/` or other normative collections.

## Series ID

`CLRP-P-YYYY-NNN`

## Lifecycle

```
Open → Under review → Accepted (becomes CLRP/amendment) | Rejected | Withdrawn
```

Accepted proposals do not remain authoritative; the merged CLRP document does.
