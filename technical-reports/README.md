# Technical reports (CLRP-TR)

Long-form methods papers, design studies, and analyses that **extend** CLRP specifications without replacing them.

## Series ID

`CLRP-TR-YYYY-NNN` — year-based sequence, independent of Synaptic Four's SF-TR genomics series.

## When to use

| Use CLRP-TR | Use CLRP-VR instead |
|-------------|---------------------|
| Simulation methodology | Psychometric pilot results |
| Architecture of conformance tests | Reliability / validity study |
| Literature synthesis | Empirical sample reporting |
| Bridge specification between programme and code | Evidence tier claims |

## Process

1. Copy [templates/technical-report-template.md](../templates/technical-report-template.md)
2. Create directory `CLRP-TR-YYYY-NNN/`
3. PR with `clrp-references` in frontmatter
4. Link from affected CLRP documents if normative impact
