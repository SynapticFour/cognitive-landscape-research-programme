# Research notes (CLRP-RN)

Short, dated, **exploratory** notes. Research notes are **not normative** unless their content is later accepted into a CLRP specification.

## Series ID

`CLRP-RN-YYYY-NNN`

## When to use

- Brainstorming dimension splits
- Pilot analysis too small for CLRP-VR
- Meeting summaries with scientific content
- Negative results not yet ready for full report

## Process

1. Copy [templates/research-note-template.md](../templates/research-note-template.md)
2. File as `CLRP-RN-YYYY-NNN-short-title.md`
3. PR — lighter review than CLRP specs

## Retention

Notes are **not deleted** when superseded; mark status in header and link forward.
